*Условия:
Программа работатет из консоли.

Для реализации функционала, при запуске скрипта необходимо анализировать командную строку
любой из библиотек разбираей параметры командной строки.


Формат запуска из консоли
tagcounter command [options]

Возможные команды:
- get
- view

При вызове команды get время запроса и название сайта должен сохранить в лог файле.
Логирование прочитаных сайтов, ведется в отдельный файл файл requests.log.
Формат файла:
дата время название_сайта

После обработки сайта получается словарь с тегами где ключ - имя тега, значение
количество тега на сайте.

Полученый словарь с тегами сохраняются в sqlite базу.
Весь словарь с тегами можно сохранять с помощью модуля pickle в одно поле базы.
Или преобразовав в JSON сохранить в виде строки.

В базу сохраняется столбцы:
- имя сайта (домен второго уровня)
- url
- дата проверки
- данные о тэгах

Для удобства работы, создать файл с синонимами в формате yaml.
synonyms.yaml

данный файл для каждого сайта опредляет синоним, формате
синоним: полное имя сайта

например:
ydx: yandex.ru
ggl: google.com

Синонимы используются в параметре get или view для ускорения ввода сайтов
Например вместо того, чтобы набирать
tagcounter --get yandex.ru
Можно обойтись синонимом
tagcounter --get ydx

Для примера, к программе приложить файл с несколькими синонимами.

Если введенная строка найдена в файле синонимов, то использовать его, иначе попробовать загружать
введенные данные как они были введены.

Пользователи сами дложны иметь возможность добавлять, удалять данные из файла синонимов.

Сделать создать unit-тесты для ключевых функций.


*Пример запуска консольной версии:
Получить список тегов:
tagcounter --get "yandex.ru"
Синонимы должны также восприниматься программой, как и обычное имя файла.
tagcounter --get "ydx"

Прочитать сохраненные данные из базы:
tagcounter --view "yandex.ru"

*Требования к оформлению пакета
Программа должна содержать файл setup.py, чтобы пакет можно было собрать с помощью команды sdist или установить с помощью pip.

После установки скрипт должен быть доступен для запуска через исполняемый файл tagcounter.
